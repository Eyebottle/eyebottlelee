## 📄 상세 설명

### 아이보틀 진료 녹음 - 진료실을 위한 스마트 녹음 솔루션

진료 중 환자와의 대화를 자동으로 녹음하고 체계적으로 관리하는 Windows 데스크톱 애플리케이션입니다.

---

### 🎙️ 주요 기능

**자동 녹음**
• 진료 시간표 기반 자동 녹음 시작/중지
• 요일별, 오전/오후 세션별 세밀한 스케줄 설정
• 10분 단위 자동 파일 분할로 관리 편의성 향상

**스마트 파일 관리**
• WAV → AAC/Opus 자동 변환으로 용량 83% 절감
  (예: 5분 녹음 WAV 30MB → AAC 5MB)
• 날짜별 폴더 자동 생성 및 체계적 정리
• 설정 가능한 보관 기간으로 자동 파일 정리
• OneDrive 연동으로 안전한 백업

**진료실 최적화**
• 마이크 자동 진단 및 입력 레벨 확인
• 볼륨 미터로 실시간 녹음 상태 모니터링
• 진료실 프로그램 자동 실행 매니저
• 시스템 트레이 상주로 백그라운드 동작

**사용자 친화적 인터페이스**
• 직관적인 3탭 구조 (대시보드, 설정, 자동 실행)
• 주간 시간표 캘린더 뷰
• 실시간 녹음 상태 및 시간 표시
• 종합 도움말 센터 내장

---

### 💡 특징

**완전 자동화**
설정 후 진료 시간에 맞춰 자동으로 녹음이 시작되고 종료됩니다. 별도 조작 없이 진료에만 집중할 수 있습니다.

**안전한 로컬 저장**
모든 녹음 파일은 사용자 PC에만 저장됩니다. 외부 서버로 전송되지 않아 환자 정보를 안전하게 보호합니다.

**용량 걱정 없음**
자동 AAC 변환으로 저장 공간을 최소화하면서도 음질은 그대로 유지합니다.

**백업 자동화**
OneDrive 연동으로 클라우드 백업을 자동화할 수 있습니다.
