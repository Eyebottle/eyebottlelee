# MS Store 스크린샷 준비 가이드

MS Store 제출에 필요한 스크린샷을 준비하는 가이드입니다.

---

## 📋 요구사항

### 기본 요구사항
- **최소 개수**: 1개 (권장: 4-5개)
- **해상도**: 1366x768 또는 1920x1080
- **형식**: PNG 또는 JPEG
- **파일 크기**: 각 50MB 이하

### 권장사항
- 실제 사용 화면 캡처
- 주요 기능이 명확하게 보이도록
- UI 요소가 선명하게 보이도록
- 개인정보나 민감한 정보 제외

---

## 🎯 필수 스크린샷 (5개)

### 1. 메인 대시보드 (녹음 상태)

**촬영 내용**:
- 녹음 대시보드 탭
- 녹음 중 상태 또는 녹음 대기 상태
- 볼륨 미터 표시
- 녹음 시작/중지 버튼

**촬영 방법**:
```
1. 앱 실행
2. "대시보드" 탭 선택
3. (선택) 녹음 시작하여 볼륨 미터 표시
4. Windows + Shift + S → 영역 선택
5. 저장: screenshot-1-dashboard.png
```

**포함되어야 할 요소**:
- ✅ 녹음 상태 카드
- ✅ 볼륨 미터 (진동하는 모습)
- ✅ 녹음 시간 표시
- ✅ 시작/중지 버튼

---

### 2. 진료 시간표 설정

**촬영 내용**:
- 녹음 설정 탭
- 주간 시간표 캘린더
- 요일별 진료 시간 설정

**촬영 방법**:
```
1. "녹음 설정" 탭 선택
2. 진료 시간표가 설정된 상태
3. 시간표 편집 다이얼로그 열기 (선택)
4. Windows + Shift + S
5. 저장: screenshot-2-schedule.png
```

**포함되어야 할 요소**:
- ✅ 주간 캘린더 그리드
- ✅ 요일별 색상 표시
- ✅ 진료 시간 텍스트
- ✅ 편집 버튼

---

### 3. 고급 설정 (WAV 변환)

**촬영 내용**:
- 고급 설정 다이얼로그
- WAV 자동 변환 옵션
- 녹음 품질 설정
- 보관 기간 설정

**촬영 방법**:
```
1. "녹음 설정" 탭
2. "고급 설정" 버튼 클릭
3. 다이얼로그가 열린 상태
4. Windows + Shift + S
5. 저장: screenshot-3-advanced-settings.png
```

**포함되어야 할 요소**:
- ✅ WAV 자동 변환 토글
- ✅ 녹음 품질 프리셋
- ✅ 보관 기간 설정
- ✅ 설명 텍스트

---

### 4. 자동 실행 매니저

**촬영 내용**:
- 자동 실행 탭
- 등록된 프로그램 목록
- 드래그 핸들
- 활성화/비활성화 스위치

**촬영 방법**:
```
1. "자동 실행" 탭 선택
2. 2-3개 프로그램 등록된 상태
   (예: 메모장, 계산기)
3. Windows + Shift + S
4. 저장: screenshot-4-auto-launch.png
```

**포함되어야 할 요소**:
- ✅ 프로그램 목록
- ✅ 드래그 핸들 (⋮⋮)
- ✅ ON/OFF 스위치
- ✅ 편집/삭제 버튼

---

### 5. 도움말 센터

**촬영 내용**:
- 도움말 다이얼로그
- 기능 설명
- 빠른 시작 가이드

**촬영 방법**:
```
1. 상단 "도움말" 버튼 (물음표 아이콘) 클릭
2. 도움말 다이얼로그 열림
3. Windows + Shift + S
4. 저장: screenshot-5-help.png
```

**포함되어야 할 요소**:
- ✅ 도움말 제목
- ✅ 섹션 리스트
- ✅ 설명 텍스트
- ✅ 아이콘/그림

---

### 6. 마이크 진단 (선택)

- 입력 레벨 표시

### 7. 저장 폴더 (선택)

- 녹음 파일 목록

## 📐 크기 및 비율

### 권장 해상도
- **1920x1080** (Full HD) - 권장
- **1366x768** (HD) - 최소

### 창 크기 조정
```
1. 앱 창을 최대화
2. 또는 고정 크기로 조정:
   - 1920x1080 화면: 창 최대화
   - 작은 화면: 창 크기 수동 조정
```

### 비율 확인
- 16:9 비율 유지
- 창이 너무 작거나 크지 않도록
- UI 요소가 잘리지 않도록

---

## 🎨 스타일 가이드

### DO ✅
- **밝은 배경** 사용 (앱의 라이트 모드)
- **실제 데이터** 사용 (진료 시간, 프로그램 이름 등)
- **깔끔한 화면** (불필요한 요소 제거)
- **선명한 텍스트** (확대해도 읽을 수 있게)

### DON'T ❌
- 개인정보 포함 (실제 진료 녹음 파일명 등)
- 오류 메시지 또는 경고 표시
- 디버그 로그나 개발자 도구
- 화면이 흐릿하거나 잘린 스크린샷

## 🚀 빠른 촬영 순서

### 한 번에 모두 촬영하기 (10분)

1. **준비**:
   ```
   - 앱 실행
   - 창 최대화
   - 진료 시간표 미리 설정
   - 자동 실행 프로그램 2-3개 등록
   ```

2. **촬영 순서**:
   ```
   1. 대시보드 → 녹음 시작 → 촬영
   2. 녹음 설정 탭 → 촬영
   3. 고급 설정 열기 → 촬영
   4. 자동 실행 탭 → 촬영
   5. 도움말 열기 → 촬영
   ```

3. **저장**:
   ```
   - 각 스크린샷을 PNG로 저장
   - 파일명 규칙에 따라 명명
   - screenshots 폴더에 정리
   ```
