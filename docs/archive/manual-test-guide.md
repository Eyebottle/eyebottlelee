# 수동 테스트 가이드 (간단 버전)

자동화 테스트 완료 후 직접 확인이 필요한 항목들입니다.

---

## 🎵 1. 음성 재생 확인 (5분)

### 테스트 방법
1. 방금 생성된 녹음 파일 찾기
   - 기본 경로: `%APPDATA%\EyebottleRecorder` 또는
   - OneDrive: `C:\Users\[user]\OneDrive\진료녹음`
2. 파일 더블클릭 또는 미디어 플레이어로 재생
3. 확인 사항:
   - [ ] 음성이 정상적으로 재생됨
   - [ ] 음질이 깨끗함 (잡음 최소)
   - [ ] 볼륨이 적절함
   - [ ] 파일이 끊김 없이 재생됨

---

## 🖥️ 2. UI 동작 확인 (10분)

### 2.1 볼륨 미터
- [ ] 녹음 시작 시 볼륨 미터가 반응함
- [ ] 말하면 미터가 올라감
- [ ] 조용하면 미터가 내려감

### 2.2 버튼 반응
- [ ] 녹음 시작 버튼 클릭 → 즉시 반응
- [ ] 녹음 중지 버튼 클릭 → 즉시 반응
- [ ] 설정 탭 전환 → 정상 작동

### 2.3 시스템 트레이
1. 앱 창 닫기 (X 버튼)
2. 확인:
   - [ ] 트레이 아이콘이 남아있음
   - [ ] 아이콘 좌클릭 → 창 복원
   - [ ] 아이콘 우클릭 → 메뉴 표시
   - [ ] "녹음 시작/중지" 메뉴 동작
   - [ ] "종료" 메뉴 → 앱 완전 종료

---

## ⚙️ 3. 시간표 자동 녹음 (15분)

### 테스트 방법
1. 앱 실행
2. "녹음 설정" 탭 클릭
3. 시간표 편집:
   - 현재 시각 + 2분 후를 진료 시작 시간으로 설정
   - 예: 지금이 14:30이면 → 14:32~15:00으로 설정
4. 저장 후 대기
5. 확인:
   - [ ] 설정한 시간에 자동으로 녹음 시작
   - [ ] 녹음 시작 시 알림/UI 변경
   - [ ] 종료 시간에 자동 중지 (선택적)

### 빠른 테스트 (대체 방법)
1. 현재 시각을 포함하도록 시간표 설정
   - 예: 지금이 14:35면 → 14:00~15:00으로 설정
2. 저장 후 약 10초 대기
3. 자동 녹음 시작 확인

---

## 🚀 4. 자동 실행 매니저 (10분)

### 4.1 프로그램 추가
1. "자동 실행" 탭 클릭
2. "프로그램 추가" 버튼
3. 테스트 프로그램 추가:
   - 메모장: `C:\Windows\System32\notepad.exe`
   - 계산기: `calc` (검색해서 추가)
4. 대기 시간: 2초
5. 확인:
   - [ ] 프로그램이 목록에 추가됨
   - [ ] 경로가 정상 표시됨
   - [ ] 스위치가 ON 상태

### 4.2 테스트 실행
1. "테스트 실행" 버튼 클릭
2. 확인:
   - [ ] 메모장이 실행됨
   - [ ] 2초 대기 후 계산기 실행됨
   - [ ] 실행된 프로그램들이 정상 작동

### 4.3 자동 실행 확인
1. 자동 실행 스위치 ON
2. 앱 종료 (완전 종료)
3. 앱 재실행
4. 확인:
   - [ ] 등록된 프로그램들이 자동으로 실행됨
   - [ ] 실행 순서가 올바름

---

## 🎨 5. 고DPI 디스플레이 (선택적, 5분)

### 테스트 방법
1. Windows 설정 → 디스플레이 → 배율
2. 150%로 변경
3. 앱 재시작
4. 확인:
   - [ ] 텍스트가 선명함
   - [ ] 버튼이 적절한 크기
   - [ ] UI가 깨지지 않음
   - [ ] 대화상자가 화면을 넘어가지 않음

---

## 📝 6. 도움말 & 튜토리얼 (5분)

### 6.1 도움말
1. 상단 "도움말" 버튼 (물음표 아이콘) 클릭
2. 확인:
   - [ ] 도움말 다이얼로그 표시됨
   - [ ] 모든 섹션이 읽기 쉬움
   - [ ] 스크롤 가능
   - [ ] 닫기 버튼 동작

### 6.2 튜토리얼
1. 앱 설정에서 튜토리얼 리셋 (가능하면)
2. 또는 첫 실행 시 확인
3. 확인:
   - [ ] 튜토리얼이 자동 표시됨
   - [ ] 3단계 모두 표시됨
   - [ ] "다음" 버튼 동작
   - [ ] "건너뛰기" 버튼 동작

---

## ⚡ 7. 에지 케이스 (선택적, 10분)

### 7.1 짧은 녹음
- [ ] 1초 녹음 → 파일 생성됨
- [ ] 30초 녹음 → WAV → AAC 변환됨

### 7.2 빈번한 중단
- [ ] 시작 → 1분 후 중지 → 시작 → 2분 후 중지
- [ ] 모든 파일이 AAC로 변환됨

### 7.3 저장 폴더 변경
1. 설정 → 저장 폴더 선택
2. 새 폴더 지정 (예: 데스크톱)
3. 녹음 실행
4. 확인:
   - [ ] 지정한 폴더에 파일 생성됨
