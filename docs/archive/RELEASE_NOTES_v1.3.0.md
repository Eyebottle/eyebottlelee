## 🎉 주요 개선사항

### 녹음 중지 시에도 자동 변환!
이제 짧은 녹음이나 중간에 녹음을 중지해도 마지막 파일이 자동으로 AAC로 변환됩니다.

**이전 (v1.2)**:
- 10분 세그먼트 분할 시에만 WAV → AAC 변환
- 짧은 녹음(1~9분)은 WAV로 남아서 용량 낭비
- 예: 5분 녹음 = 30 MB (WAV)

**개선 (v1.3)**:
- 녹음 중지 시에도 자동 변환 ✨
- 모든 녹음 파일이 AAC로 변환되어 용량 절감
- 예: 5분 녹음 = 5 MB (AAC) - **83% 절감!**

### 공식 로고 적용
eyebottle.kr의 공식 로고가 앱에 적용되었습니다.

---

## ✨ 새로운 기능

### WAV → AAC 완전 자동화
- 10분 세그먼트 분할 시: 자동 변환 (기존 기능)
- 녹음 중지 시: 마지막 파일도 자동 변환 (신규!)
- 변환 지연: 5초 대기 후 변환 시작
- 원본 삭제: 변환 성공 시 WAV 파일 자동 삭제

**혜택:**
- 💾 용량 절감: WAV 30MB → AAC 5MB (83% 절감)
- 🎯 빈번한 중단에도 최적화
- 📦 모든 녹음이 일관되게 AAC 형식
