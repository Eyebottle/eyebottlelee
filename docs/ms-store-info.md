## 📝 참고사항

### Store ID 관리
- Store ID는 **절대 변경되지 않습니다**
- 앱을 삭제하고 다시 등록해도 새로운 ID가 부여됩니다
- 이 ID를 안전하게 보관하세요

### Package Identity
- pubspec.yaml에 정확히 입력해야 합니다
- 한 글자라도 다르면 업로드 실패합니다
- 업데이트 시에도 동일하게 유지해야 합니다

### Deep Link 활용
```dart
// Flutter에서 MS Store 열기 예시
import 'package:url_launcher/url_launcher.dart';

Future<void> openMsStore() async {
  final uri = Uri.parse('ms-windows-store://pdp/?ProductId=9NDZB0QSL928');
  if (await canLaunchUrl(uri)) {
    await launchUrl(uri);
  }
}
```
