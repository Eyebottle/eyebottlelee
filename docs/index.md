## 🎯 주요 기능

### 자동 녹음
- 진료 시간표 기반 자동 녹음 시작/중지
- 요일별, 오전/오후 세션별 세밀한 스케줄 설정
- 10분 단위 자동 파일 분할

### 스마트 파일 관리
- WAV → AAC/Opus 자동 변환으로 용량 83% 절감
- 날짜별 폴더 자동 생성 및 체계적 정리
- 설정 가능한 보관 기간으로 자동 파일 정리
- OneDrive 연동 안전한 백업

### 진료실 최적화
- 마이크 자동 진단 및 입력 레벨 확인
- 볼륨 미터로 실시간 녹음 상태 모니터링
- 진료실 프로그램 자동 실행 매니저
- 시스템 트레이 상주로 백그라운드 동작

## 💻 시스템 요구사항

- **운영체제**: Windows 10 버전 1809 이상
- **프로세서**: x64
- **메모리**: 4GB RAM (권장: 8GB)
- **저장공간**: 100MB (앱) + 녹음 파일 저장 공간
- **장비**: 마이크 장치
