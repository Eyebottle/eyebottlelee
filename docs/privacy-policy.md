# 개인정보 처리방침

**아이보틀 진료 녹음 (Eyebottle Medical Recorder)**

**최종 업데이트:** 2025년 11월 7일
**발효일:** 2025년 11월 7일

---

## 1. 개요

아이보틀 진료 녹음("본 앱")은 사용자의 개인정보 보호를 최우선으로 생각합니다. 본 개인정보 처리방침은 본 앱이 어떤 정보를 수집하고, 어떻게 사용하며, 어떻게 보호하는지를 설명합니다.

---

## 2. 수집하는 정보

### 2.1 마이크 입력 (오디오 녹음)

**수집 목적:**
- 진료 중 대화 녹음

**수집 방법:**
- 사용자가 녹음 시작 버튼을 클릭하거나 자동 녹음 스케줄이 실행될 때
- Windows 마이크 권한 필요

**저장 위치:**
- 사용자가 지정한 로컬 폴더
- 기본: 문서 또는 OneDrive 폴더

**파일 형식:**
- AAC (Advanced Audio Coding)
- 파일명: `YYYYMMDD_HHmmss_진료녹음.aac`

### 2.2 앱 설정 정보

**수집 목적:**
- 사용자 설정 저장 및 로드

**수집 항목:**
- 녹음 저장 경로
- 진료 시간표 설정
- 녹음 품질 설정
- 보관 기간 설정
- 자동 실행 프로그램 목록

**저장 위치:**
- 로컬 앱 데이터 폴더
- `%LOCALAPPDATA%\Packages\eyebottle.medical.recorder_*\LocalState\`

---

## 3. 정보 사용 목적

본 앱은 수집한 정보를 다음 목적으로만 사용합니다:

1. **녹음 기능 제공**
   - 마이크 입력을 녹음 파일로 저장
   - 자동 파일 변환 (WAV → AAC)

2. **사용자 설정 유지**
   - 진료 시간표 저장 및 자동 녹음 스케줄링
   - 앱 재시작 시 설정 복원

3. **파일 관리**
   - 날짜별 폴더 자동 생성
   - 보관 기간에 따른 자동 파일 삭제

---

## 4. 정보 공유 및 제공

### 4.1 제3자 제공

**본 앱은 어떠한 정보도 제3자에게 제공하지 않습니다.**

- 녹음 파일은 로컬에만 저장
- 앱 설정은 로컬에만 저장
- 외부 서버로 데이터 전송 없음
- 분석 또는 광고 목적의 데이터 수집 없음

### 4.2 OneDrive 백업 (선택적)

사용자가 OneDrive 폴더를 녹음 저장 경로로 선택한 경우:

- 녹음 파일이 OneDrive로 자동 동기화될 수 있음
- OneDrive는 Microsoft의 서비스로, Microsoft 개인정보 처리방침이 적용됨
- 사용자가 직접 제어 가능 (OneDrive 동기화 끄기/켜기)

**참고:** OneDrive 사용은 완전히 선택적이며, 다른 로컬 폴더를 선택할 수 있습니다.

---

## 5. 정보 저장 및 보안

### 5.1 저장 위치

**녹음 파일:**
- 사용자가 지정한 로컬 폴더
- 사용자가 직접 접근 및 관리 가능

**앱 설정:**
- Windows 앱 데이터 폴더 (로컬)
- 다른 앱과 격리된 샌드박스 환경

### 5.2 보안 조치

- **로컬 저장:** 모든 데이터는 사용자 PC에만 저장
- **암호화:** Windows 파일 시스템 암호화 지원 (BitLocker 등)
- **권한 관리:** Windows 사용자 계정 권한으로 접근 제어
- **자동 삭제:** 설정된 보관 기간에 따라 자동 파일 정리

### 5.3 데이터 보관 기간

**녹음 파일:**
- 사용자가 설정한 보관 기간
- 기본: 90일
- 사용자가 직접 삭제 가능

**앱 설정:**
- 앱 제거 시까지 보관
- 앱 제거 시 자동 삭제됨

---

## 6. 사용자 권리

사용자는 다음 권리를 가집니다:

### 6.1 접근 권리
- 저장된 녹음 파일에 언제든 접근 가능
- Windows 탐색기로 직접 확인 가능

### 6.2 수정 권리
- 앱 설정을 언제든 수정 가능
- 저장 경로, 스케줄 등 자유롭게 변경

### 6.3 삭제 권리
- 녹음 파일을 직접 삭제 가능
- 앱 제거 시 모든 설정 자동 삭제

### 6.4 거부 권리
- 마이크 권한을 거부할 수 있음 (녹음 기능 사용 불가)
- 자동 녹음 스케줄을 비활성화할 수 있음

## 7. 마이크 권한

### 7.1 권한 요청

본 앱은 다음 권한을 요청합니다:

**마이크 (필수)**
- 목적: 오디오 녹음
- 거부 시: 녹음 기능 사용 불가

**파일 시스템 접근 (자동 허용)**
- 목적: 녹음 파일 저장 및 관리
- 사용자가 선택한 폴더에만 접근

### 7.2 권한 관리

사용자는 Windows 설정에서 언제든 권한을 변경할 수 있습니다:

```
Windows 설정 → 개인정보 → 마이크
→ "아이보틀 진료 녹음" 찾기
→ 토글 켜기/끄기
```
