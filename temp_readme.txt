아이보틀 진료 녹음 v1.3.0 - 테스트 릴리즈
==========================================

📋 버전 정보
-----------
버전: 1.3.0
빌드 번호: 11
빌드 날짜: 2025-11-06

🆕 이번 버전의 주요 변경사항
-------------------------
✅ 시간표 자동 녹음 버그 수정
   - 요일 매핑 오류 해결
   - 설정한 시간표대로 정확하게 녹음 시작/중지

✅ 스케줄 리셋 로직 개선
   - 시간표 변경 시 즉시 반영
   - 구 스케줄과 신 스케줄 중복 실행 문제 해결

✅ 저장기간 자동 정리 기능 검증 완료
   - 1주일/1개월/3개월 등 설정 가능
   - 오래된 녹음 파일 자동 삭제

🧪 테스트 항목
-------------
□ 녹음 기능 (WAV 폴백 + 자동 변환)
□ 시간표 자동 녹음 (정확성 확인)
□ 자동 실행 매니저
□ 저장 위치 설정
□ 마이크 진단

⚙️ 설치 방법
-----------
1. eyebottlelee-v1.3.0-release 폴더 전체를 진료실 PC로 복사
2. medical_recorder.exe 실행
3. Windows SmartScreen 경고 시: "추가 정보" → "실행"

⚠️ 중요: 시간표 재설정 필요
-----------------------
이전 버전에서 업그레이드하는 경우:
→ 설정 탭 > 진료 시간표에서 모든 요일을 다시 확인하고 저장해주세요.
→ 요일 매핑 버그 수정으로 인해 기존 설정이 1씩 밀려있을 수 있습니다.

📁 폴더 구조
-----------
medical_recorder.exe      - 실행 파일
data/                     - 앱 데이터 (ffmpeg.exe 포함)
*.dll                     - 필요한 라이브러리들

🐛 문제 발생 시
-------------
1. 마이크 진단 카드 > "진단 정보" > "진단 정보 복사"
2. 로그 위치: %USERPROFILE%\Documents\EyebottleRecorder\logs\
3. 로그 파일을 수집하여 전달

💡 주요 로그 확인 방법
------------------
앱 실행 후 다음 로그를 확인:

📅 새로운 진료 시간표 적용 시작
  일요일: 휴무
  월요일: 09:00~13:00, 14:00~18:00
  ...
✅ 새로운 진료 시간표 적용 완료

진료 시간 확인: 요일=X(요일명), 시각=HH:MM
  -> 요일 스케줄: ...
  -> 진료 시간 여부: true/false

📞 문의
------
문제가 발생하면 로그와 함께 연락주세요.
