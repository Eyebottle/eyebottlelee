# MS Store 스크린샷 폴더

MS Store 제출용 스크린샷을 저장하는 폴더입니다.

---

## 📋 필요한 스크린샷 (5개)

### 1. screenshot-1-dashboard.png
- **내용:** 메인 대시보드 (녹음 상태)
- **촬영:** 녹음 중 또는 녹음 대기 상태
- **포함:** 볼륨 미터, 시작/중지 버튼, 녹음 시간
- **해상도:** 1920x1080 권장

### 2. screenshot-2-schedule.png
- **내용:** 진료 시간표 설정
- **촬영:** 주간 캘린더 그리드
- **포함:** 요일별 색상, 진료 시간 텍스트, 편집 버튼
- **해상도:** 1920x1080 권장

### 3. screenshot-3-advanced-settings.png
- **내용:** 고급 설정 다이얼로그
- **촬영:** "고급 설정" 버튼 클릭 후
- **포함:** WAV 변환 토글, 녹음 품질, 보관 기간
- **해상도:** 1920x1080 권장

### 4. screenshot-4-auto-launch.png
- **내용:** 자동 실행 매니저
- **촬영:** 자동 실행 탭
- **포함:** 프로그램 목록, 드래그 핸들, ON/OFF 스위치
- **해상도:** 1920x1080 권장

### 5. screenshot-5-help.png
- **내용:** 도움말 센터
- **촬영:** 도움말 버튼 (?) 클릭 후
- **포함:** 도움말 제목, 섹션 리스트, 설명 텍스트
- **해상도:** 1920x1080 권장

---

## 📐 요구사항

**MS Store 스크린샷 규격:**
- **최소 해상도:** 1366x768
- **권장 해상도:** 1920x1080
- **비율:** 16:9
- **형식:** PNG 또는 JPEG
- **파일 크기:** 각 50MB 이하
- **최소 개수:** 1개
- **권장 개수:** 4-5개

---

## 🎨 촬영 가이드

**준비:**
1. 앱 실행 및 창 최대화
2. 진료 시간표 미리 설정
3. 자동 실행 프로그램 2-3개 등록

**촬영 도구:**
```
Windows + Shift + S (화면 캡처 도구)
→ 영역 선택
→ 자동으로 클립보드에 복사됨
→ 그림판 등에 붙여넣기 후 저장
```

**파일 저장:**
```
이 폴더(screenshots/)에 저장
파일명: screenshot-1-dashboard.png 형식
```

**상세 가이드:**
```
docs/screenshot-guide.md 참조
```
